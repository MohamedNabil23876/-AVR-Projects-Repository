# 🔌 AVR Projects Repository

This repository contains a collection of **AVR embedded systems projects** that were developed as part of my work and teaching experience.

## 📚 Overview

The projects in this repository are a mix of:

* Personal implementations
* Practical examples developed during teaching sessions
* Student projects from embedded systems scholarships

All projects are built using **AVR microcontrollers** and focus on applying real embedded concepts.

---

## 🛠️ Technologies & Concepts

* Embedded C
* AVR Microcontrollers (e.g., ATmega series)
* GPIO, Timers, Interrupts
* PWM, ADC
* UART, SPI, I2C
* Real-time problem solving

---

## 🎓 Teaching Approach

These projects were used to help students:

* Understand embedded systems practically
* Apply theoretical concepts in real scenarios
* Build complete mini-systems step by step

Each project is designed to be:

* Simple and clear
* Focused on a specific concept
* Easy to extend and modify

---

## 🚀 Project Examples

* LED Control Systems
* Smart Home Modules
* Sensor Interfacing
* Communication Systems (UART / SPI / I2C)
* Automation mini-projects

---

## 🎯 Purpose

The goal of this repository is to:

* Showcase practical embedded systems work
* Provide learning references for students
* Demonstrate real teaching experience in embedded systems
